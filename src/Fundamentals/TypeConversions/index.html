Type Conversions

        - Convert object from one type to another

        
String Conversions

        - Convert object of another type to String
        - alert() converts anything inside of it to String
        - Explic conversion requires "String()". Ex: String(wordToConvertToString)


        
Numeric Conversions

        - Convert object to number
        - Explicit conversion is possible. 

        Ex: 
        let str = "123"
        let num = Number(str)

        Numeric Conversion Rules:
        Value           Becomes
        undefined           Nan
        null                0
        true/false          1/0
        string           WhiteSpaces are removed. Number is read from String. Error gives Nan


        
Boolean Conversions

        - Converts to Boolean

        Conversion Rule:
        - You will get false for: Empty String, 0, null, undefined, NaN
        - Else you will get True

        Note:
        - String with "0" or " " (Empty Spaces is String) is true
