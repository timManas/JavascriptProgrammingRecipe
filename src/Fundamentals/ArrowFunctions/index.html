Arrow Functions

    What ? 
    - Shorter way of writing Function expressions
    - Quicker way of writin one-liner actions 
    - Very very similar to Anonymous functions

    Syntax:     
    let func = (arg1, arg2, arg3) => expression

    Equivalent of:
    let func = function(arg1, arg2, arg2) {
        return expression
    }

    Example:
    let sum = (a, b) => a + b

    Equivalent of:
    let sum = function(a, b) {
        return a + b
    }

    
MultiLine Arrow functions

    What ?
    - Arrow expresisons but contain more lines

    How ?
    - Just add "{}" after the => 

    Example:
    let sum = (a,b) => {
        let result = a + b
        return result
    }

    alert(sum(1,2))
